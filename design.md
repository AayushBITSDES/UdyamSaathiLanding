# Udyam Saathi - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Teal (#0F4C5C) - Trust, stability, government authority
- **Secondary**: Sage Green (#7A9B76) - Growth, support, accessibility
- **Accent**: Warm Amber (#D4A574) - Innovation, warmth, approachability
- **Neutral**: Charcoal Gray (#2D3748) - Professional text
- **Background**: Soft Cream (#FEFCF7) - Clean, accessible background

### Typography
- **Headings**: Sofia Pro Narrow Semi Bold
  - H1: 2.5rem (40px) - Hero headlines
  - H2: 2rem (32px) - Section headers
  - H3: 1.5rem (24px) - Subsection headers
- **Body**: General Sans Regular
  - Body: 1rem (16px) - Main content
  - Small: 0.875rem (14px) - Captions, metadata
- **Line Height**: 1.6 for body text, 1.3 for headings

### Visual Language
- **Minimalist Government Tech**: Clean, trustworthy, and professional aesthetic
- **Accessibility-First**: High contrast ratios, clear hierarchy, touch-friendly interactions
- **Voice-First Innovation**: Subtle audio/voice visual cues without overwhelming
- **Indian Context**: Warm, approachable design that feels local yet modern

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions and page transitions
- **ECharts.js**: Clean data visualization for problem gap chart
- **Typed.js**: Typewriter effect for hero headline
- **Splitting.js**: Letter-by-letter text animations for emphasis
- **p5.js**: Subtle background particle system representing connectivity

### Effects & Animations
- **Hero Section**: 
  - Typewriter animation for "Udyam Saathi" headline
  - Subtle fade-in for description text
  - Floating particle background representing data connectivity
- **Problem Gap Visualization**: 
  - Animated bar chart showing scheme access disparity
  - Progressive disclosure on scroll
- **Feature Cards**: 
  - Gentle hover lift effect with shadow
  - Staggered reveal animation on page load
- **Navigation**: 
  - Smooth underline transition for active states
  - Mobile hamburger menu with slide animation

### Styling Approach
- **Grid-Based Layout**: Consistent 12-column grid system
- **Generous Whitespace**: Breathing room around content sections
- **Card-Based Components**: Clean containers for features and information
- **Subtle Shadows**: Soft elevation without heavy drop shadows
- **Rounded Corners**: 8px radius for modern, friendly feel

### Header Effect
- **Clean Navigation Bar**: Fixed top navigation with subtle background blur
- **Hero Background**: Abstract geometric composition with soft gradients
- **Text Hierarchy**: Clear contrast between headline, subheadline, and description
- **Early Prototype Badge**: Prominent but non-intrusive label

### Responsive Design
- **Mobile-First**: Optimized for small screens and touch interactions
- **Breakpoints**: 
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+
- **Touch Targets**: Minimum 44x44px for all interactive elements
- **Typography Scaling**: Responsive font sizes maintaining readability

### Accessibility Features
- **Color Contrast**: 4.5:1 minimum ratio for all text
- **Focus Indicators**: Clear 3px outline for keyboard navigation
- **Semantic HTML**: Proper heading hierarchy and landmark elements
- **Alt Text**: Descriptive text for all meaningful images
- **Screen Reader Support**: ARIA labels and descriptions where needed
